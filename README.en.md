# 人事员工管理系统SSM1

#### Description
基于SSM的人事员工管理系统共有两种角色，分别为管理员和员工，功能如下：

* 管理员：部门管理、员工管理、职位管理、公告管理、文档管理

* 员工：查看部门、员工、职位、公告等信息
